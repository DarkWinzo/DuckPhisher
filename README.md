### INSTALLATION [ TERMUX APP --ANDROID ]
* git clone https://github.com/DarkWinzo/DuckPhisher/
* cd DuckPhisher/
* chmod 777 start.sh
* ./start.sh
* ./Is-DuckPhisher.sh

### INSTALLATION [ KALI ]
* git clone https://github.com/DarkWinzo/DuckPhisher/
* cd DuckPhisher/
* chmod 777 setup.sh
* Bash setup.sh
* Bash DuckPhisher.sh
